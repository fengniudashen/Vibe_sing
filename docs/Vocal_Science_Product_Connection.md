## 1. 人声发声的物理学基础

### 1.1 声音的产生：三层系统

人声是一个**三级级联系统**：

```
第一层：动力源（肺 → 气流）
  ↓ 呼出的气流推动声带
第二层：振动源（声带 → 基频 F0）
  ↓ 声带振动产生初始声波（含丰富谐波）
第三层：共鸣腔（咽腔+口腔+鼻腔 → 调制频谱）
  ↓ 声道的形状"雕刻"出不同的音色
最终输出：我们听到的声音
```

这三层任何一层出问题，都会导致特定的"声学签名"——Vibesing 就是通过捕捉这些签名来诊断问题。

### 1.2 基频（F0）与泛音

- **基频 F0**：声带每秒振动的次数（单位 Hz）。决定我们感知的"音高"。
  - 男声说话约 80-180 Hz
  - 女声说话约 150-300 Hz
  - 高音 C5 = 523 Hz，高音 C6 = 1046 Hz

- **泛音列（Harmonics）**：声带振动不是纯正弦波，而是同时产生 F0 的整数倍频率（2F0, 3F0, 4F0...）。这些泛音的相对强度决定了声音的"音色"和"质感"。

**Vibesing 关联**：Mel 频谱图（80 维输入特征）本质上就是在捕捉这些泛音的能量分布。不同的发声方式会产生完全不同的泛音列形状。

### 1.3 谐噪比（HNR）

声音中同时存在两种成分：
- **谐波（Harmonic）**：声带规律振动产生的有规律成分 → 听起来"清亮"、"干净"
- **噪声（Noise）**：声带闭合不完全或不规律振动产生的无规律成分 → 听起来"沙哑"、"毛糙"

$$
\text{HNR} = 10 \cdot \log_{10}\left(\frac{\text{谐波能量}}{\text{噪声能量}}\right) \quad \text{(dB)}
$$

- HNR > 20 dB = 非常干净的声音
- HNR 12-20 dB = 正常范围
- HNR < 12 dB = 声音含大量噪声 → **可能是挤卡或漏气**

**Vibesing 关联**：HNR 是 V0.1 挤卡检测的**核心特征**。当声带过度紧张（挤卡）或过度松弛（漏气），HNR 都会骤降。

## 2. 声区与发声类型

### 2.1 四种基础声区

| 声区 | 英文 | 声带状态 | 音域范围 | 声学特征 |
|------|------|---------|---------|---------|
| **胸声** | Chest Voice | 声带全长振动、厚重接触 | 低音到中音 | 低频泛音强、H1-H2 较负 |
| **头声** | Head Voice | 声带边缘振动、轻薄接触 | 中高音到高音 | 高频泛音丰富、H1-H2 接近 0 |
| **假声** | Falsetto | 声带极薄振动、不完全闭合 | 高音以上 | 基频突出、泛音极少、HNR 低 |
| **咽音** | Pharyngeal / Twang | 声道窄化产生的共鸣增强 | 全音域 | 3kHz 附近能量集中（Singer's Formant） |

### 2.2 声区的物理本质

声区的本质是**声带振动模式**的离散跳变（register transition）。想象声带是一根可调截面的弦：

```
胸声：弦全长振动（全振幅） ════════════════════
                              ↓ 换声点 (Passaggio)
头声：弦边缘振动（部分振幅） ═══─ ─ ─═══─ ─ ─═══
                              ↓
假声：弦极薄振动（表面波）    ─ ─ ─ ─ ─ ─ ─ ─ ─ ─
```

### 2.3 换声点（Passaggio）

每个人有 1-2 个换声点——声带振动模式被迫切换的音高。

- 男声第一换声点：~E4 (330 Hz)
- 女声第一换声点：~E5 (660 Hz)

**换声点附近是"挤卡"和"破音"的重灾区**——因为声带在两种振动模式之间"拉锯"，如果肌肉控制不好，就会代偿。

**Vibesing 关联**：MVP 的 F0 > 220Hz 高音增益逻辑，本质上是在检测用户是否处于换声点附近。

## 3. 混声——高音的圣杯

### 3.1 什么是混声？

混声（Mix Voice）不是一种独立的声区，而是**胸声和头声共振比例的精确控制**。

```
纯胸声 ◄─────────── 混声 ──────────────► 纯头声

胸声比例  100%  80%  60%  50%  40%  20%  0%
头声比例    0%  20%  40%  50%  60%  80%  100%
            │         │         │         │
          低音区    换声点附近  中高音    极高音
```

完美的混声意味着：从低音到高音，胸声和头声的比例**平滑过渡**，听众完全感觉不到声区切换。

### 3.2 为什么混声是"高音的圣杯"？

| 发声方式 | 高音效果 | 问题 |
|---------|---------|------|
| 纯胸声硬唱到高音 | 爆发力强但极容易挤卡/破音 | 声带负荷过重，损伤风险 |
| 换到假声 | 安全但声音突然变虚变弱 | 听众会感知到明显的"断层" |
| **混声** | **既有力量又轻松，高音听起来毫不费力** | **需要精确的肌肉协调，是声乐训练的终极目标** |

### 3.3 混声的声学签名

| 指标 | 纯胸声 | 纯头声/假声 | 混声 |
|------|--------|-----------|------|
| H1-H2 (第一谐波-第二谐波差) | 很负 (-6~-12 dB) | 接近 0 或正 | 中间值 (-2~-6 dB) |
| 频谱斜率 (Spectral Tilt) | 陡峭下降 | 平坦 | 中等 |
| Singer's Formant (3kHz) | 弱 | 弱 | **强**（混声训练良好时） |
| HNR | 高 (>20dB) | 中等 (假声较低) | 高 (>18dB) |

**Vibesing 关联**：V0.2 的混声检测就是基于以上特征差异，将当前声音分类为 5 档（纯胸/胸混/平衡/头混/纯头）。

## 4. 发声病灶——高音路上的地雷

### 4.1 挤卡（Vocal Squeeze / Constriction）

**Vibesing V0.1 的核心检测目标。**

#### 物理机制

```
正常发声：
  声带 → 自然内收 → 规律振动
  假声带（室带）→ 保持分开，不参与

挤卡：
  声带 → 过度内收（拉得太紧）
  假声带（室带）→ 也开始内收、挤压声门上方
  喉室（Morgagni's ventricle）→ 塌陷
  
  结果：声门上方压力激增 → 声带振动不规律 → 大量噪声成分注入
```

#### 听感

"憋"、"卡"、"挤"、"紧"——像是声音被一只手掐住了。

#### 声学签名

| 指标 | 正常 | 挤卡 |
|------|------|------|
| HNR | > 18 dB | **< 12 dB** ← Vibesing 核心判据 |
| CPP | > 8 dB | **< 5 dB** |
| 频谱 1-3kHz | 平滑过渡 | **异常隆起/鼓包**（假声带振动叠加） |
| 声门接触商 (CQ) | 0.4-0.6 | **> 0.7**（过度闭合） |

#### 好发场景

- 换声点附近的高音（E4-G4 男声，B4-D5 女声）
- 音量突然增大时（用力喊高音）
- 紧张/怯场时

#### Vibesing 如何帮助

用户挤卡时 → 红灯亮起 → 用户意识到"我在挤了" → 有意识地放松 → 红灯灭掉。这个"即时因果反馈"是声乐老师上课时做的事——Vibesing 让这件事在练习时也能发生。

---

### 4.2 大白嗓 / 喊叫（Yelling / Untrained Belting）

#### 物理机制

声带全力闭合 + 大气量猛推，但**没有声道共鸣的配合**。就像用蛮力推开一扇门，而不是巧妙地转动门把手。

#### 声学签名

| 指标 | 效果 |
|------|------|
| H1-H2 | **极负**（声门闭合过强） |
| Singer's Formant (3kHz) | **完全缺失** ← 与"训练有素的 Belting"的核心区别 |
| 频谱 | 1-4kHz 能量"摊平"，无聚焦 |
| 整体音量 | 极大 |

#### 区别：大白嗓 vs 训练有素的 Belting

专业歌手的 Belting（如 Whitney Houston, Adele）听起来也很大声，但他们通过咽腔窄化产生了强烈的 Singer's Formant（~3kHz），声音穿透力极强但声带负荷可控。大白嗓没有这个共鸣增强，纯靠蛮力。

---

### 4.3 喉位高（High Larynx）

#### 物理机制

唱高音时，喉头（甲状软骨）不自觉地随音高升高。这会导致**声道缩短**。

声道缩短 → 所有共振峰频率上移 → 声音变"尖"、"细"、"紧"。

```
正常（喉头中位）：
  声道长度 ~17cm (男性)
  F1 ≈ 500 Hz, F2 ≈ 1500 Hz, F3 ≈ 2500 Hz

喉位高：
  声道长度 ~14cm（缩短 ~18%）
  F1 ≈ 600 Hz, F2 ≈ 1800 Hz, F3 ≈ 3000 Hz
  所有共振峰上移 ~18%
```

#### 声学签名

关键区别：喉位高时 F1/F2/F3 是**同步、等比例上移**的（因为声道整体缩短），而挤卡不是。

#### Vibesing 关联

V1.0+ 通过检测共振峰偏移的一致性来区分喉位高和挤卡。

---

### 4.4 漏气 / 闭合不严（Breathiness / Glottal Insufficiency）

#### 物理机制

声带闭合不完全，气息从缝隙泄出 → 大量噪声成分混入声音。

#### 声学签名

| 指标 | 正常 | 漏气 |
|------|------|------|
| HNR | > 18 dB | **< 10 dB** |
| CPP | > 8 dB | **< 4 dB** |
| 声门闭合商 (CQ) | 0.4-0.6 | **< 0.3** |

**注意**：漏气和挤卡在 HNR 上都表现为"低值"，但原因相反——挤卡是"过度闭合+假声带参与"，漏气是"闭合不全+气息泄出"。区分二者需要 H1-H2（挤卡时很负，漏气时接近 0 或正值）。

---

### 4.5 气息不稳（Breath Instability）

#### 物理机制

横膈膜支撑不足 → 声门下气压波动 → 声音振幅不规律抖动。

#### 声学签名

- 帧间能量变异系数 (CV) 异常高
- F0 微颤（非艺术性颤音，不规律的 pitch wobble）

## 5. 共鸣与声道物理学

### 5.1 共振峰（Formants）

共振峰是声道的固有共振频率。声道的形状决定了哪些频率被增强。

```
F1 (第一共振峰) ← 主要由口腔开合度决定
  张嘴大 → F1 高（如"啊"的 F1 ≈ 800Hz）
  闭嘴 → F1 低（如"衣"的 F1 ≈ 300Hz）

F2 (第二共振峰) ← 主要由舌位前后决定
  舌头靠前 → F2 高（如"衣"的 F2 ≈ 2300Hz）
  舌头靠后 → F2 低（如"乌"的 F2 ≈ 800Hz）

F3 (第三共振峰) ← 主要由咽腔形状决定
  咽腔窄化 → F3 与 F4 靠拢 → Singer's Formant（~3kHz）
```

### 5.2 四种共鸣位置

| 共鸣类型 | 声道特征 | 声学签名 | 听感 |
|---------|---------|---------|------|
| **面罩共鸣** (Mask Resonance) | 声波聚集在双颊/前额 | F3/F4/F5 能量增强，高频丰富 | "亮"、"集中"、"穿透力强" |
| **鼻音过重** (Excessive Nasality) | 软腭下降太多，鼻腔过度参与 | 出现反共振峰(anti-formant) + F1 能量异常 | "闷"、"鼻塞" |
| **口腔音/扁塌** (Oral / Flat) | 口腔形状扁平，无前后纵深 | F1/F2 分布平淡，无聚焦 | "扁"、"白" |
| **喉音** (Laryngeal) | 过度收缩咽腔/喉室 | 颤抖的低频成分 + 高频衰减 | "闷"、"压抑" |

**Vibesing 关联**：V1.0+ 的共鸣位置检测，通过分析 F3-F5 频段能量分布模式来分类。

## 6. 修饰技巧与时序动态

### 6.1 换声断层 / 破音 (Voice Break / Crack)

声带振动模式在瞬间被迫切换 → F0 出现 >50Hz 的帧间跳变。

**Vibesing 关联**：V1.0+ 用纯规则检测（F0 帧间差 > 阈值），不需要深度学习。

### 6.2 颤音 (Vibrato)

艺术性的 F0 和振幅的周期性调制。

```
正常颤音：
  F0 调制频率：4.5-6.5 Hz
  F0 调制深度：0.5-2 个半音
  振幅调制同步
  
不良颤音：
  喉颤音（Wobble）：频率 < 4 Hz，深度过大
  摇音（Tremolo）：频率 > 7 Hz，深度过大
```

**Vibesing 关联**：V1.0+ 通过 F0 的短时频率分析（对 F0 轨迹做 FFT）来检测和评估颤音质量。

### 6.3 气泡音 (Vocal Fry)

声带以极低频率（20-80 Hz）不规律振动。

**Vibesing 关联**：可用规则检测（F0 < 80Hz + Jitter 极高），不需要深度学习。

## 7. 隐蔽代偿动作

这些是"用错误的方式唱对了音高"——听起来可能还行，但长期会造成声带损伤，且阻碍混声的建立。

### 7.1 舌根后压 (Tongue Root Retraction)

#### 机制

用户不自觉地把舌根往后压，试图"打开空间"——但这实际上是在压缩咽腔后部。

#### 声学签名

F2 异常降低（因为舌根后移等效于口腔后部扩大，拉低 F2）。同一元音在正常发声 vs 舌根后压时，F2 会差 200-400 Hz。

#### 为什么难检测

需要知道用户正在发哪个元音（不同元音有不同的"正常 F2"），才能判断当前 F2 是否"异常低"。这意味着需要元音识别能力。

**Vibesing 关联**：V2.0 通过轻量 ASR 子模型识别元音，再对比实际 F2 与该元音的标准 F2。

### 7.2 下巴僵硬 (Jaw Tension)

#### 机制

下巴肌肉紧张，导致口腔开合不灵活，辅音和元音之间的过渡变得"拖沓"。

#### 声学签名

辅音→元音的过渡时间（Transition Duration）异常长（>80ms，正常应 <50ms）。

## 8. Vibesing 如何"听到"这一切

### 8.1 声乐老师 vs Vibesing 的"感知映射"

| 声乐老师的判断 | 大脑处理过程（隐式） | Vibesing 的技术等价 |
|-------------|------------------|------------------|
| "你在挤！" | 听到声音变"紧"、"卡" | HNR 骤降 + 频谱噪声增加 |
| "你在喊！" | 听到声音变"粗"、"白" | H1-H2 极负 + Singer's Formant 缺失 |
| "喉头太高了" | 听到声音变"尖"、"细" | F1/F2/F3 系统性上移 |
| "漏气了" | 听到声音变"虚"、"散" | HNR 骤降 + H1-H2 异常偏正 |
| "混声找到了！" | 听到声音"轻松"且"有力" | 频谱能量分布均衡 + HNR 高 + Singer's Formant 出现 |
| "那个音破了" | 听到音高瞬间跳变 | F0 帧间差 > 50Hz |
| "有点鼻音" | 听到声音变"闷" | 反共振峰出现 |

### 8.2 为什么"实时"如此重要

声乐老师的核心教学方式：

```
学生唱 → 老师 [立刻打断] → "停！刚才那个音你挤了" → 学生调整 → 老师 "对了！就是这个感觉"
```

这个**即时打断+即时确认**的反馈循环，是声乐教学的精髓。

如果 Vibesing 的延迟 > 200ms，用户会感觉反馈和自己的发声"对不上"，因果关系建立失败，产品价值归零。

这就是为什么我们将端到端延迟硬性限制在 < 100ms——让红灯亮起的瞬间，用户**还在发那个"有问题的"声音**。

### 8.3 技术实现层次

```
层次 1：物理规则（V0.1）
  if HNR < 12 dB → 红灯
  优点：零延迟训练数据需求，可立刻上线
  缺点：无法区分"挤卡"和"漏气"（都是 HNR 低）

层次 2：单任务深度学习（V0.2）
  CNN 学习 Mel 频谱中的微妙模式
  优点：能区分 HNR 低的不同原因
  缺点：需要几千条标注数据

层次 3：多任务深度学习（V1.0）
  双流架构同时输出 7+ 维诊断
  优点：全面诊断
  缺点：需要上万条多维标注数据

层次 4：个性化模型（V3.0）
  基于个人历史数据微调
  优点：适应每个人的嗓音特点
  缺点：需要大量个人数据积累
```

### 9.2 核心产品闭环

```
声乐物理学原理
    ↓ 提取关键声学指标
DSP 特征（HNR/CPP/F0/Mel/共振峰）
    ↓ 模式识别
AI 模型 → 多维诊断结果
    ↓ 平滑 + 可视化
UI 反馈（红绿灯 / Slider / 评分）
    ↓ 用户感知并调整
发声改善 → 下次更好 → 高音觉醒 ✨
```

### 9.3 为什么 Vibesing 不只是"另一个 K 歌 App"

| 维度 | K 歌 App | Vibesing |
|------|---------|---------|
| 检测的是什么 | **音准**（你唱的音对不对） | **发声质量**（你这个音是怎么发出来的） |
| 类比 | "你投篮投进了吗？" | "你的投篮姿势对不对？关节有没有受力不当？" |
| 能否帮用户进步 | ❌ 知道跑调但不知道为什么 | ✅ 知道挤卡+知道是哪里挤+知道怎么调整 |
| 技术栈 | F0 (YIN/CREPE) | Mel + HNR + CPP + F0 + 共振峰 + CNN |
| 竞品壁垒 | 几乎为零 | 声学特征工程 + 端侧 AI + 数据飞轮 |

---

> **结论**：Vibesing 的每一个功能都锚定在一个明确的物理声学原理上。这不是玄学，不是经验主义的"感觉判断"——而是**可量化、可复现的物理信号检测**。声乐老师用经验和直觉做的事，Vibesing 用数学和 AI 做到了同样的效果，而且可以 24/7 陪在用户身边。
